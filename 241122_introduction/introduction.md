### ターミナルを使う
ディレクトリの作成
```
mkdir 241122_introduction
```
ファイルの作成
```
touch 241122_introduction/nodetrial.js
```

カレントディレクトリの移動
```
cd 241122_introduction
```

node.js環境の確認
```
node -v
// v20.17.0
```

### JavaScriptを実行する
カレントディレクトリに注意！（241122に移動済）
```
node nodetrial.js
```
